# Assets Folder

This folder contains your personal assets for the portfolio website.

## Required Files

Please add the following files to this folder:

1. **profile-photo.jpg** - Your profile photo
   - Recommended size: 400x400 pixels or larger (square aspect ratio)
   - Supported formats: .jpg, .jpeg, .png, .webp

2. **resume.pdf** - Your resume/CV
   - Format: PDF
   - File name: resume.pdf
